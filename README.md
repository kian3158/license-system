# License System

Dev repo for License Manager (Go) + License Client (Python).

Structure:
- `manager-go/` — Go license manager server
- `client-python/` — Python client skeleton
- `hw-emulator/` — dev hardware-emulator (FastAPI)
- `docs/` — architecture, API schemas, design decisions
- `.github/workflows/` — CI pipelines

See `docs/ARCH.md` for architecture and design choices.
